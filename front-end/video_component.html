Play.LimitlessTCG Twitch Extension

This extension was created by Dillonzer with a huge shout-out to Robin and the LimitlessTCG team for creating Play.LimitlessTCG and allowing myself to access the data.

How to Use

The extension is hopefully self explanatory but there are some hidden features.

  • The data currently does not auto-refresh due to wanting to limit loads on the Server. To refresh, please click the little refresh button near the Round Timer.
  • The data will also refresh everytime you go to a new area of the Extension (ie. Clicking on Standings / Match Information)
  • The little icon near the Streamer Name indicates if the streamer is still playing in this tournament of if they have dropped.
  • The little icon near the Tournament Name indicates if the tournament is currently ongoing or not.

Thanks for using the extension and for using Play.LimitlessTCG!
