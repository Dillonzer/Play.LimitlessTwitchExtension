Play.Limitless Twitch Extension

This extension was created by Dillonzer with a huge shout-out to Robin and the Limitless TCG team for creating Play.Limitless and allowing myself to access the data.

How to Use

The extension is hopefully self explanatory but there are some hidden features.

  • If the streamer changes the tournament on their end, you can click the Limitless Logo in the top right to refresh the page.
  • The little icon near the Streamer Name indicates if the streamer is still playing in this tournament of if they have dropped.
  • The little icon near the Tournament Name indicates if the tournament is currently ongoing or not.

Thanks for using the extension and for using Play.Limitless!
